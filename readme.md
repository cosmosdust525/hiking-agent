# 🥾 徒步路线规划-我的AI助手

一个基于 AI 的徒步路线规划工具，帮助用户规划徒步路线并生成互动地图。

## 功能特点

- 🔍 智能路线规划：输入起点和终点，自动规划最佳徒步路线
- 🗺️ 互动地图展示：生成可交互的 HTML 地图
- 🤖 AI 行程建议：使用大模型生成个性化徒步计划
- 📊 路线详情：显示距离、累计爬升、预计时间等

## 技术栈

- Python 3.11
- Streamlit - Web 界面
- OSMnx - 路网数据
- Folium - 地图可视化
- Ollama - 本地大模型
- deepseek - cloud大模型
